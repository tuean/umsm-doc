### 促销商品列表

#### 功能

促销商品列表

#### 请求地址

```text
/api/product/discount/list
```

#### 请求方式

POST

#### 请求头

token-wx 或 token-pc 或 token-pc-ad

#### 请求参数

```json
{
    "pageSize": 20,
    "pageNum": 1,
    "storeId": "10000",
    "searchKey": null
}
```
