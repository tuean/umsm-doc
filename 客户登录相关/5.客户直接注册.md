
### 客户直接注册

#### 功能
未注册客户注册账户
返回当前客户的token

#### 请求地址
```text
/api/login/register/direct
```

#### 请求方式
POST

#### 请求参数
```json
{
  "wxCode": xx, // wx.getLogin 获取到的一次性code 必填
  "wxName": xx, // 微信名
  "avatar": xx // 头像
  "doorId": xx // 门id 非必填
}
```

#### 返回结果
1. 正常
```json
{
  "code": 0,
  "message": "successs",
  "body": {
      "token": "xxxx" // 系统长期token
  }
}
```

2. 失败
```json
{
  "code": 1,
  "message": "error message"
}
```
