
### 新增门店

#### 功能
新增门店

#### 请求地址
```text
/api/store/info/add
```

#### 请求方式
POST

#### 请求头
token-pc 或 token-pc-ad 或者 token-wx

#### 请求参数
```json
{
  "storeName":"门店1"  // 门店名
}
```

#### 返回结果
1. 正常
```json
{
  "code":0,
  "message":"ok",
  "data":null
}
```
