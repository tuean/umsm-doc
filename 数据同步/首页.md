### 首页信息

#### 功能
获取首页信息

#### 请求地址
```text
/api/data/store/banner
```

#### 请求方式
GET

#### 请求参数
url param
```text
storeId  店铺id
```

#### 返回结果

```json
{
    "code":0,
    "message":"ok",
    "data":[
        "123.png",
        "456.png"
    ]
}

```

#### curl
```
curl --request GET \
  --url 'http://localhost:7777/api/data/store/banner?storeId=10000'
```
