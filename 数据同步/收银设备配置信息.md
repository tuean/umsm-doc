### 配置信息

#### 功能
设定一些配置

#### 请求地址
```text
/api/data/checkout/setting
```

#### 请求方式
GET

#### 请求参数
无

#### 返回结果

```json
{
	"code": 0,
	"message": "ok",
	"data": {
		"checkout_pay_success_info": "消息提示", // 收银设备成功支付显示说明
		"checkout_auto_open_door": "false", // 收银成功后是否自动开门 是：true 否：false
		"checkout_button_expire": "15", // 收银设备按钮倒计时时间（秒）
		"checkout_pay_show": "消息提示" // 打印小票提示
	}
}
```
